# Módulo 01 - Gerenciamento de Controle de Acesso

Projeto oficial desenvolvido por Eliana Almeida.

## Estrutura oficial do módulo

Este módulo está localizado na raiz deste repositório e utiliza os seguintes arquivos e pastas:

- app.py
- requirements.txt
- database/
- static/
- templates/

## Funcionalidades implementadas

- Login de acesso ao sistema
- Logout de sessão
- Dashboard premium
- CRUD de usuários
- CRUD de módulos do sistema
- CRUD de ambientes/prédios
- Banco de dados em JSON

## Como executar

```bash
pip install -r requirements.txt
python app.py
